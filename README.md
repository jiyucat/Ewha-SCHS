# Ewha-SCHS

# 🎵 국제음악치료포럼 웹사이트 (SCHS International Music Therapy Forum)
이 프로젝트는 **SCHS**에서 주최한 **국제음악치료포럼**의 웹사이트입니다.  
음악 치료 분야의 최신 연구 및 발표를 공유하고, 전 세계 전문가들이 교류할 수 있도록 제작되었습니다.

## 📌 주요 기능
- **홈페이지:** 포럼 소개 및 개요 제공
- **세션 일정:** 강연 및 워크숍 일정 확인 가능
- **연사 소개:** 발표자 및 패널 명단 제공
- **참가 신청:** 온라인 등록 및 신청 폼
- **게시판:** 참가자 간의 소통 공간

## 🛠 기술 스택
- **Frontend:** HTML, CSS, JavaScript
- **CSS Framework:** Tailwind CSS / Bootstrap
- **JavaScript Framework:** React.js / Vue.js (사용한 경우)
- **Backend:** Firebase / Node.js (사용한 경우)
- **Deployment:** Vercel / Netlify / GitHub Pages
